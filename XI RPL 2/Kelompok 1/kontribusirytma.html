Cara penggunaan Git Checkout

  • <nama-cabang> — Beralih ke cabang lokal tertentu.
  • -b <cabang baru> — Membuat dan langsung beralih ke cabang baru.
  • -b <cabang baru> --track <cabang jarak jauh> — Membuat cabang baru dengan pelacakan ke cabang remote.
  • <jalur-berkas> <hash-komit> — Memulihkan versi lama dari berkas tertentu.

Contoh Penggunaan

$ git checkout other-branch

Menjadikan other-branch sebagai cabang aktif.

$ git checkout -b new-branch

Membuat dan langsung beralih ke new-branch.

$ git checkout -b new-branch --track origin/develop

Membuat cabang lokal baru yang melacak origin/develop.

$ git checkout 8a7b201 index.html

Memulihkan index.html dari commit 8a7b201.

$ git checkout HEAD index.html

Reset index.html ke versi terakhir yang dikomit.
