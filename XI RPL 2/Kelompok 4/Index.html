Saya berperan dalam pembuatan slide presentasi, dimulai dari pembuatan desain, pengumpulan materi, hingga penyusunan slide presentasi. Selain itu, saya juga adalah orang yang mencoba mengimplementasikan dan menjalankan kode git branch dan git merge untuk di praktekkan langsung oleh saya sendiri.

git Branch

git branch -> untuk melihat daftar branch yang ada di repository. git branch nama_branch -> untuk membuat branch baru dengan nama "nama_branch". git checkout nama_branch -> untuk berpindah ke branch "nama_branch".

git status

Untuk menampilkan status file di repository seperti: File baru (untracked), File yang berubah tapi belum di-stage, dan File yang sudah siap di-commit.

git log

Berfungsi untuk melihat riwayat commit secara lengkap seperti siapa yang commit, kapan, dan pesan commit.

git merge (nama_branch)

berfungsi untuk menggabungkan perubahan dari branch bernama (nama_branch) ke branch yang sedang aktif saat ini.

git --merged

Perintah ini digunakan untuk menampilkan daftar branch yang sudah digabungkan (merged) ke branch saat ini.
